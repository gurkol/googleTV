# Google TV Projekty

Kolekce her a aplikací pro Google TV.

## Projekty

### River Rider

Pixel art hra inspirovaná klasickou hrou River Raid.

- **Typ**: HTML5 hra v Android WebView
- **Složka**: `riverRider/`
- **Ovládání**: D-pad na Google TV dálkovém ovladači

Více informací v [riverRider/README.md](riverRider/README.md)

## Build

Každý projekt obsahuje vlastní Gradle build konfiguraci. Pro sestavení APK:

```bash
cd riverRider
./gradlew assembleDebug
```
